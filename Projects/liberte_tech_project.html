Info

This work was a big challenge for me because I need to collaborate with this great agency to enhance their new site's UX. In this project, UX takes a huge role and it matters a lot especially for an IT agency like this. When I saw the older site's design and UX it looks so bad to me personally and It needs to be much better than the previous one. Then I got the opportunity to redesign it so I tried my best to make it as better as possible.

Process

First of all I do full research about this project that how I can improve it more and the target audience is always the aim. Here I tried to be consistent with the users experience more. Where the interactions is required & where the CTA’s are necessary etc. Then I created wireframes (low fidelity designs) for every single page with responsive UX. So the UI designer will be more comfortable in his work.
